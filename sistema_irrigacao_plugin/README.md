# Sistema de Irrigação Plugin

Um plugin profissional iterativo de QGIS para auxiliar na orçamentação e elaboração da capacidade hídrica de tubos de agronegócio e aspersão.

## Instalação
1. Com o QGIS aberto, vá em **Configurações > Opções > Sistema** e identifique a pasta do perfil ativo.
2. Acesse a pasta `python/plugins/` (crie se não existir).
3. Copie esta pasta inteira (`sistema_irrigacao_plugin`) para o diretório `plugins/`.
4. Reinicie o QGIS.
5. Acesse o menu **Complementos > Gerenciar e Instalar Complementos**.
6. Ative o **Sistema Dimensionamento Irrigação**.

## Passos da Ferramenta

O sistema processa 18 passos divididos desde mapeamento de zona UTM até orçamentação final.
